About The Mod
=============

This mod brings the Alpine A390, Alpine's 100% electric sport fastback, into
Assetto Corsa as a complete, immediately playable package.

This version is intended for players equipped with Content Manager and Custom
Shaders Patch. It includes the car model, its driving physics, sounds, liveries,
preview visuals, as well as the graphical enhancements required for a modern
experience in Assetto Corsa.

Alpine and Renault names, logos, trademarks, vehicle designs and other identity
elements remain the exclusive property of their respective owners.


Requirements
============

- Assetto Corsa: Steam 64-bit version, v1.16.3 or v1.16.4.
- Content Manager: v0.8.2594.39678 or later recommended.
- Custom Shaders Patch: v0.2.11 minimum mandatory.
- Recommended CSP version: v0.2.11 public release, or a more recent preview if
  you already use one.
- Recommended installation method: drag and drop into Content Manager.


Installation
============

Install Assetto Corsa
---------------------

- Purchase and install Assetto Corsa via Steam.
- Launch the game once so that it creates all the necessary folders.

Install Content Manager And Custom Shaders Patch
------------------------------------------------

- Download and install Content Manager.
- From within Content Manager, install Custom Shaders Patch (CSP).

Mod Installation - Recommended Method: Content Manager
------------------------------------------------------

- Download the complete Alpine A390 mod archive.
- Open Content Manager.
- Drag and drop the archive into the Content Manager window.
- Click Install when Content Manager detects the package.
- Search for "Alpine A390" in the cars list.
- Select the vehicle, choose a livery, make sure that Extended Physics is
  enabled, and start your session.

Manual Method
-------------

- Extract the contents of the archive.
- Copy the entire mc_alpine_A390 folder to:

  assettocorsa/content/cars/

- Launch Assetto Corsa or Content Manager.
- Search for "Alpine A390" in the cars list.

The expected final path is:

assettocorsa/content/cars/mc_alpine_A390

If the car does not appear, the folder may have been mistakenly placed inside
another directory. When in doubt, reinstall the complete archive via Content
Manager.


Included Content
================

- Playable Alpine A390 GTS car.

Controls

Overboost
---------

Content Manager > Settings > Assetto Corsa > Controls > Patch > Extra option A

Hold the button while throttle is above 80% to activate Overboost.

Troubleshooting
===============

The Car Does Not Appear In The Game
-----------------------------------

- Check that it is properly installed in:

  assettocorsa/content/cars/mc_alpine_A390

- Restart Content Manager.
- If needed, reinstall the complete archive.

No Sound
--------

- Reinstall the complete archive.
- Make sure the installation completed without interruption.

The Game Crashes On Loading
---------------------------

- Update Custom Shaders Patch.
- Reinstall the complete archive.
- Check the Content Manager logs to identify the exact error.

Lights, Wipers, Mirrors Or Materials Not Working Properly
---------------------------------------------------------

- Update Custom Shaders Patch.
- Check your CSP settings in Content Manager.
- Run a new test after a clean reinstallation.

Missing Or Incorrect Textures
-----------------------------

- Reinstall the complete archive.
- Never combine files from different versions of the mod.
